Процесс обучения GCNN состоит из следующих шагов:

Подготовка данных: Первый шаг заключается в подготовке данных, которые будут использоваться для обучения модели. Данные должны представляться в виде графа. Например, если мы используем GCNN для классификации узлов в социальной сети, то каждый узел представляет отдельного пользователя, а ребра представляют связи между пользователями.

Создание модели: Следующим шагом является создание модели GCNN. Она состоит из нескольких слоев, в которых применяются операции свертки над графами. Каждый слой GCNN может быть настроен на выбор определенных параметров, таких как количество фильтров, размерность выходного пространства или функция активации.

Обучение модели: Во время обучения GCNN используется стандартный метод градиентного спуска для минимизации функции потерь. Эта функция определяет, насколько хорошо модель соответствует правильным ответам. В GCNN для каждого слоя используется свой граф-сигнал, который передается через узлы сети.

Оценка качества модели: После завершения процесса обучения, модель оценивается на тестовых данных. Это помогает определить качество модели и ее способность обобщаться на новые данные.

Использование модели: После того, как модель успешно обучена, она может быть использована для анализа новых графов. Например, если мы создали GCNN для классификации пользователей в социальной сети, мы можем использовать ее для классификации новых пользователей на основе их связей с другими пользователями.
